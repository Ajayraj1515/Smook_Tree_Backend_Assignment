# User and Address Registration Backend

This is a simple Node.js application that allows users to register with their name and address. The application uses MySQL for data storage and Sequelize ORM to manage the database models.

## Technologies Used

- **Node.js**: Backend runtime
- **Express.js**: Web framework
- **Sequelize**: ORM for MySQL
- **MySQL**: Relational database
- **HTML**: Simple form for frontend

## Features

- Register users with their name and address.
- One-to-many relationship between users and addresses.
- Data persistence in MySQL database.
